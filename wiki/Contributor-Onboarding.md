# Contributor Onboarding

## Prerequisites

- .NET SDK
- Node.js + Yarn
- Git

## Fast path

1. Read `README.md`.
2. Read `MIGRATION_PLAN.md`.
3. Read `CONTRIBUTING.md`.
4. Build backend and frontend.
5. Pick an issue labeled `good-first-issue` or `help-wanted`.

## Recommended first contribution areas

- Documentation improvements.
- Metadata provider test scaffolding.
- API contract and validation helpers.
- CI workflow polish.

## Collaboration conventions

- Keep PRs focused and reviewable.
- Add tests/docs with functional changes.
- Use labels and project fields consistently.
